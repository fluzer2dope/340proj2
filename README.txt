---< Project Description: >---
The included files, once compiled with 'make', will start a console to allow accessing a floppy disk (or image).

---< Compiling Instructions: >---
From the command line, run 'make clean' and then 'make'. Once the program has compiled, run 'flop' to begin accepting data.
If desired, 'make debug' will provide the necessary symbols to gdb to debug this code.
